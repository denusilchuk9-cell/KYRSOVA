# Електронний журнал

![CI/CD Pipeline](https://github.com/denusilchuk9-cell/KYRSOVA/actions/workflows/dotnet.yml/badge.svg)

## Опис
Десктопний застосунок для обліку студентів, оцінок та відвідування.

## Технології
- Windows Forms .NET Framework 4.8.1
- C#
- MSTest для юніт-тестування

## Функціонал
- Додавання/видалення студентів
- Додавання/видалення предметів
- Виставлення оцінок
- Відмітка відвідування
- Розрахунок середнього балу
- Розрахунок відсотка відвідуваності

## Тести
- 30 юніт-тестів
- Покриття коду: ~50%

## CI/CD
Автоматична збірка та тестування через GitHub Actions.
